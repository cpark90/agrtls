# 안테나 딜레이 캘리브레이션 기록

모듈 개체별 + 칩별로 Adelay 값을 기록한다. 기본값(DW1000:16384 / DW3000:16385)을
프로덕션에 쓰지 않는다. 실측값은 칩별 rf_config 헤더에 반영.

| 장치 ID | 역할 | 칩 | 모듈 | 거리(m) | Adelay | 측정일 | 비고 |
|---|---|---|---|---|---|---|---|
| ANCHOR_01 | Anchor | DW1000 | UWB Pro | 8 | | | |
| TAG_01 | Tag | DW1000 | UWB Pro | 8 | | | |
| ANCHOR_01 | Anchor | DW3000 | UWB DW3000 | 8 | | | |

절차: 고정거리(8m+) 배치 → 캘리브레이션 스케치 → Adelay 기록 → rf_config 반영 →
재측정으로 오차감소 확인. Pro/일반/칩 간 값 재사용 금지.
